## ⏰ 프로젝트 회고

### 👍 좋았던 점

- QWidget을 사용하여 윈도우창의 title, ico, style 등 다뤄보는 방법을 습득할 수 있었습니다.
- 'pyuic6 -x 파일명.ui -o 파일명.py' 명령어를 통해 ui파일을 py파일로 변환하는 방법을 습득할 수 있었습니다.
- DB에 데이터를 삽입, 조회 그리고 로그인 처리를 해보는 방법을 습득할 수 있었습니다.
